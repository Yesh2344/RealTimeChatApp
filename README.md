## Introduction
Real-Time Chat Application is a production-ready, advanced Bash project that utilizes WebSocket support for real-time communication. The application is designed with proper error handling, logging, and comments to ensure maintainability and scalability.

## Installation
To install the Real-Time Chat Application, follow these steps:
1. Clone the repository using Git: `git clone https://github.com/username/repo.git`
2. Navigate to the project directory: `cd repo`
3. Install dependencies: `bash install.sh`
4. Configure the environment variables: `cp .env.example .env`

## Usage
To start the application, run the following command: `bash main.sh`

## API Documentation
The Real-Time Chat Application API provides the following endpoints:
- `/connect`: Establishes a WebSocket connection
- `/send`: Sends a message to all connected clients
